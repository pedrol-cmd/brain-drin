# Email Sequence Generator

## Purpose
To build outbound engines that get replies by focusing on relevance and value.

## Steps
1. **Signal Selection**: Identify the trigger for the sequence (e.g., New post, Hiring, Tech stack change).
2. **Hook Engineering**: Write 3-5 personalized hooks for the first touch.
3. **Narrative Arc**: Define the follow-up flow (Value → Social Proof → Contrast → Ask).
4. **A/B Variation**: Generate variants for subject lines and body copy.

## Rules
- Max 80 words per email.
- One clear CTA.
- No "Hope you're doing well" fluff.
