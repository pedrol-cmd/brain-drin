# Onboarding Experience Sync

## Purpose
To ensure that new users understand the core value proposition as quickly as possible, driving long-term retention.

## Metrics
- **Time to Aha**: How long until the primary value is realized.
- **Completion Rate**: % of users who finish the setup.
- **D1 Retention**: % of users who return after day 1.

## Actions
1. **Audit**: Walk through the flow as a total stranger.
2. **Identify Friction**: Where do users stop/hesitate?
3. **Simplify**: Can we remove steps or defer them?
4. **Delight**: Where can we add value before asking for effort?

## Rules
- Don't ask for permission before providing value.
- Clear over clever.
