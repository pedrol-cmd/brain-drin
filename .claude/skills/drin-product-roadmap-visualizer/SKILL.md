# Product Roadmap Visualizer

## Purpose
To maintain a high-level, visual alignment across the organization regarding future product direction.

## Formats
- **Timeline**: `gantt` based roadmaps for specific quarters.
- **Thematic**: `mindmap` for grouped strategic pillars.
- **Kanban Style**: `journey` for feature flow progress.

## Workflow
1. **Intake**: Gather prioritized epics from the backlog.
2. **Group**: Categorize by theme or goal.
3. **Visualize**: Generate the Mermaid code.
4. **Share**: Export to README or project docs.

## Rules
- Focus on outcomes, not just dates.
- Keep it high-level (Epics, not stories).
