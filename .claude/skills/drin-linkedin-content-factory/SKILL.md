# LinkedIn Content Factory

## Purpose
To build personal and company brand authority through consistent, high-value LinkedIn presence.

## Workflow
1. **Topic Extraction**: Pull 3 ideas from a recent blog post, PRD, or conversation.
2. **Hook Engineering**: Generate 5 variations of the first 2 lines.
3. **Structure Match**: Use proven formats (The Story, The List, The Contrarion).
4. **Engagement Prep**: Write 3 questions for the comments section.

## Rules
- High density.
- No AI-isms.
- Vertical layout for mobile readability.
